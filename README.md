# Book Library Application

## Overview
This project is a simple book library application that allows users to manage their personal library. Users can add books, filter them based on their reading status, and store the data in the browser's local storage.

## Project Structure
```
book-library-app
├── src
│   ├── css
│   │   └── styles.css        # Styles for the application
│   ├── js
│   │   ├── script.js # Manages library functionality and local storage
│   │   └── libro.js          # Defines the structure of a book object
│   └── index.html            # Main HTML document for the application
└── README.md                 # Documentation for the project
```

## Features
- Add books with title, author, and genre.
- Filter books by reading status (all, read, unread).
- Store book data in local storage for persistence.

## Technologies Used
- HTML
- CSS
- JavaScript
